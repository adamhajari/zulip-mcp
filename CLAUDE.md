# Zulip MCP — Claude Code Instructions

## Anonymizer mapping file

The file `~/.config/zulip_mcp/anonymizer_map.json` is **off-limits**.

- Do NOT read, display, or reference its contents under any circumstances.
- Do NOT use it to de-anonymize aliases ({User1}, {User2}, etc.) found in messages or summaries.
- De-anonymization is handled externally by a separate script — it is never Claude's job.
